# PrivySVG - 浏览器端PNG转SVG工具

PrivySVG 是一个完全在浏览器本地运行的 PNG/JPG 转 SVG 矢量图工具。所有图片处理均在用户设备上完成，不会上传到任何服务器，充分保护用户隐私。

## 特性

- 🔒 **完全本地处理**：所有转换在浏览器中完成，图片永不离开设备
- 🎨 **多种路径模式**：支持像素、多边形、样条三种矢量化风格
- ⚡ **实时预览**：上传后立即预览转换效果
- 📱 **响应式设计**：适配桌面和移动设备
- 🌓 **色彩模式**：支持彩色和黑白两种模式
- ⚙️ **可调参数**：提供多种参数精细调整转换效果

## 技术栈

- HTML5 + CSS3 + JavaScript (ES6+)
- Vite 构建工具
- Canvas API 图像处理
- SVG 矢量图形

## 快速开始

### 开发环境

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build
```

### 部署

构建后的 `dist` 目录可以部署到任何静态托管服务：
- GitHub Pages
- Vercel
- Netlify
- Cloudflare Pages

## 使用说明

1. **上传图片**：点击或拖拽图片到上传区域，支持 PNG、JPG、WebP、BMP、GIF 格式
2. **调整参数**：
   - 路径模式：选择像素、多边形或样条风格
   - 色彩模式：彩色或黑白
   - 其他参数：色彩精度、过滤斑点、渐变步长等
3. **下载SVG**：点击下载按钮保存转换后的SVG文件

## 隐私说明

所有图片处理均在浏览器本地完成：
- 图片不会被上传到任何服务器
- 不需要网络连接即可使用（离线可用）
- 转换完成后不会保留任何数据
